# mirai-ws-bot

go 实现的基于 mirai-api-http 的 bot 框架。

目前是破烂，不要用。

## 目录结构

- bot 主体框架
	- api mirai-api-http 的接口
	- connect 与mirai-api-http 建立websocket连接
	- logging 简易日志
- config 配置文件读取
- modules 用户模组
